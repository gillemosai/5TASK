# Manual de Instalação e Uso - 5Task

Este guia orienta como configurar e rodar o aplicativo **5Task** em um ambiente Windows de forma automática.

## 1. Instalação Automática

Para facilitar, criamos um instalador que faz tudo sozinho.

1.  Abra a pasta do projeto **5Task**.
2.  Encontre o arquivo **`Instalar Tudo.bat`**.
3.  Clique com o **botão direito** nele e selecione **"Executar como Administrador"**.
4.  Uma janela preta abrirá e instalará automaticamente:
    *   Node.js (se não tiver)
    *   Python (se não tiver)
    *   Todas as bibliotecas necessárias
    *   O atalho na sua pasta

*Aguarde o processo terminar. Pode demorar alguns minutos dependendo da sua internet.*

## 2. Como Usar

Após a instalação, um novo atalho chamado **`5Task App`** foi criado.

1.  Dê um duplo-clique no **`5Task App`**.
2.  O sistema abrirá automaticamente.
3.  **Para sair**: Basta fechar a janela do aplicativo. O servidor desligará sozinho.

---
**Solução de Problemas:**
*   *Pediu permissão de administrador?* Sim, clique em "Sim" para permitir a instalação dos programas.
*   *Deu erro de Winget?* Certifique-se de que está no Windows 10 ou 11 atualizado.
*   *O App não abre?* Reinicie o computador para garantir que as instalações foram finalizadas.
